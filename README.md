# bloodlab-common
Enums and constants used in multiple libraries and services

###### Install
`go get github.com/blutspende/bloodlab-common`

## Encoding
List of encodings. Can be used with `github.com/blutspende/go-astm` or directly to match with `golang.org/x/text/encoding/charmap` charmaps.

## Timezone
List of timezones. Can be used for `time.LoadLocation` locations.

## MessageType
List of message types. Used throughout bloodlab to identify and process messages.

## MessageStatus
List of message statuses. Used to store and read states of messages.
